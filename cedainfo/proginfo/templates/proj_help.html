Projects are activities supported by programmes. They have specific science aims.

The tool here is designed to record a minimum information to scope and track the project. Below are the attributes needed and what they mean.
Attribute nameMeaningValue in for this project
Grant A grant number or reference. This should be the lead split for a multi-institute grant. {{ proj.grant }}
Title The full title or long name of the project. {{ proj.title }}
Desc A short description of the project. This can come from the grants on the web system. {{ proj.desc|truncatewords:30 }}
StartDate The start date of the project. {{ proj.Startdate }}
EndDate The end date of the project. {{ proj.Enddate }}
PI The Principle investigatior. The person responsible for the project as a whole. {{ proj.sciSupContact }}
CoI1 A Co-Investorgator. This is the project contact who can act as deputy for the PI. {{ proj.CoI1 }}
CoI2 A Co-Investorgator. As CoI1. {{ proj.CoI2 }}
Cost The financal size of the project in k. This gives us an idea of the scale and how many people we need to talk to. {{ proj.cost }}
Thirdpartydata Data requested by the project to help the project work. For example, data from the Met Office. {{ proj.Thirdparydata|truncatewords:30 }} ...
Services Projects may need some development work or other things that will incur extra costs. These should be brifly described here. {{ proj.Services }}
Status The current status of the project.
  • Not scoped: A project has only some simple details like the PI name and title.
  • Scoping: Acquiring details and listing data products.
  • Unresponsive: The PI has not responded to the scoping questions (see notes on contacting PIs).
  • Active: The project has been included in an SC approved DMP.
  • Complete: All data are archived or there is agreement that no data is expected. The project must also be over to be complete.
 {{ proj.state }}
Prog This is the link to the funding programme supporting the project. {{ proj.prog }}
Unresponsive If the PI and Co-I's contacts are unresponsive then this flag should be ticked. {{ proj.Unresponsive }}
Data products This is the list of assosiated data products. To add another data product click on "Add data products". To edit an existing data products click on items from the list.
Notes Add notes about the project by typing text in the box and clicking "Add note". Text starting with http:// will appear as links, text starting with Q followed by a number will be a displayed as a link to a query (e.g. Q17), and text starting with a # then a number (e.g. #5000) links to a CEDA ticket.
